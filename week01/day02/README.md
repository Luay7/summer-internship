# Day 02: Prompting Fundamentals

## Today's Goal
The goal of today's task is to master the basics of Prompt Engineering and observe how applying its principles across progressive levels enhances the AI's output performance.

## Task 1: Practical Implementation of Prompts
**What I did:**
* Learned the core elements of writing an effective prompt (Task, Role, Context, Format, Constraints).
* Created a Python script (`main.py`) to practically apply this knowledge.
* Wrote 5 different prompts aiming to solve the exact same problem (explaining "Prompt Engineering"), applying prompt engineering step-by-step to improve the prompt's efficiency at every single level.
* Used a `for` loop to send these prompts sequentially to the Gemini model and printed the results to observe the raw outputs.

**Key Takeaway:**
I discovered that when a prompt is highly detailed, clear, and structured, the AI's output becomes much more accurate, precise, and specific. This clearly demonstrates the true advantage and power of prompt engineering.
